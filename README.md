## 🔍 What is gptop?

gptop is a simple tool that shows you how your computer’s GPU is doing. It works on both Apple Silicon and NVIDIA graphics cards. This app uses a text-based interface you can see in the command window.

You can watch your GPU’s status like temperature, memory use, and performance all in one place. This helps you make sure your GPU is working well and not getting too hot.

gptop works on many systems, but this guide focuses on how to get it running on Windows.

## 💻 How to Use gptop

When gptop is running, you will see several fields, such as:

- **GPU Name:** The model of your graphics card.  
- **GPU Temperature:** How hot your GPU is in degrees Celsius.  
- **Memory Usage:** Amount of GPU memory currently in use.  
- **GPU Load:** Percentage of GPU power being used.  
- **Fan Speed:** The speed of your GPU fan (if supported).  

The numbers update automatically every few seconds. This helps you monitor GPU health and performance without needing complex software.

## 🔧 Tips for Better Monitoring

- Keep gptop running while gaming or using graphics programs to check GPU stress.  
- If GPU temperature goes above 85°C, consider improving your PC’s cooling.  
- Watch the memory use to avoid running out during heavy tasks.  
- You can resize the command window to see more data clearly.

## ✅ Troubleshooting

- **Nothing shows after running:** Make sure your GPU driver is up to date.  
- **Error says GPU not found:** Check if your PC’s GPU is supported and properly installed.  
- **Command window closes immediately:** Run the `.exe` file from Command Prompt manually to see error messages.  
  - Open Command Prompt by typing `cmd` in the Windows search bar.  
  - Navigate to the folder where gptop is saved using the `cd` command.  
  - Type `gptop.exe` and press Enter.
